# create-kiva

kiva系列模板快速搭建工具

## 获取方法

```
npm install create-kiva -g
create-kiva install react-kiva
```

## kiva系列模板开发

1. 发布模块包中必须包含template文件夹
2. template文件夹中可以包含gitignore文件，注意不要加.，模板安装时自动添加
3. 命名规则 <自定义名称>-kiva
